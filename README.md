> A simple cli tool to display files with not-so-good-looking colourful output.

  

  



  

  

## Install

  

  

```sh

  
git clone https://github.com/roshanlc/reader.git
cd reader
make && sudo make install
make clean
```

## Uninstall
```sh
sudo make uninstall
```

## Usage

  

  

```sh

reader [FILE-NAME]/[options]
  
Example:
reader ~/.bashrc
reader -u
reader -v
```

  

  

### Supports
* Piped input
* Displaying files
